# Email Campaign Analysis (SQL, BigQuery)

This project analyzes email campaign performance and user activity using SQL in Google BigQuery.

## Overview
The goal of this analysis is to evaluate email engagement and identify top-performing countries and user segments.

## Key Features
- Calculated email funnel metrics (sent, open, visit)
- Aggregated data by country and user attributes
- Combined multiple datasets using joins and CTEs
- Applied window functions to rank top-performing countries

## Metrics
- Number of sent emails
- Open rate
- Visit rate
- Number of active accounts

## Tools
- SQL (BigQuery)

## Example Analysis
- Top 10 countries by email volume
- Top 10 countries by number of accounts
- Email engagement across different user segments

## File Structure
- `queries/email_analysis.sql` — main SQL query for analysis
